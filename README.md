# 智能选股工具MVP

金融自动化创业项目 - 标准化产品 · 闭环自动化 · 睡后收入

## 项目概述

这是一个完整的智能选股工具MVP，实现了：
- **标准化产品**：封装专业选股能力为标准化Web工具
- **闭环自动化**：数据获取→筛选→报告→发布全自动运行
- **睡后收入**：用户自助使用，系统自动交付，无需人工介入
- **自然流量获客**：通过高质量免费工具吸引精准用户
- **比赛验证**：可用此工具参加创业比赛，证明产品价值

## 功能特性

1. **实时数据**：集成东方财富实时股票行情接口
2. **智能筛选**：支持价格、涨跌幅、成交量、PE、ROE、市值等多维度筛选
3. **自动刷新**：每10分钟自动刷新数据，后台线程持续运行
4. **报告生成**：支持CSV和JSON格式报告下载
5. **专业界面**：响应式设计，专业金融工具UI
6. **RESTful API**：完整的API接口，支持第三方集成

## 技术架构

- **后端**：Python Flask，RESTful API设计
- **前端**：HTML5/CSS3/JavaScript，响应式设计
- **数据源**：东方财富实时行情接口
- **部署**：支持云平台一键部署（Railway、Heroku、Vercel等）

## 快速开始

### 本地运行

```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py

# 访问 http://localhost:5000
```

### 云部署（Railway）

1. 在Railway.app注册账号
2. 连接GitHub仓库
3. 导入此项目
4. 自动部署完成

### 云部署（Heroku）

```bash
# 安装Heroku CLI
heroku login
heroku create your-app-name
git push heroku main
heroku open
```
