# 知乎文章创作提示词：高流量高转化率智能选股工具文章

## 核心目标
撰写一篇在知乎平台上能够获得高流量（阅读量10万+）和高转化率（工具访问量1000+）的智能选股工具推广文章。

## 受众画像
1. **A股散户投资者**：资金量较小，缺乏专业分析工具，渴望免费实用的投资辅助工具。
2. **量化投资初学者**：对数据驱动投资感兴趣，但编程能力有限。
3. **价值投资者**：关注基本面分析，寻找低估值股票。
4. **技术分析爱好者**：关注价格走势和成交量等技术指标。

## 内容核心要点
### 1. 痛点共鸣（开头部分）
- 股票太多难以筛选
- 基本面研究耗时耗力
- 付费工具昂贵，免费工具功能简陋
- 缺乏系统化的筛选方法

### 2. 解决方案引入
- 介绍智能选股工具作为“AI投资助手”的定位
- 强调“完全免费”、“无广告”、“开源透明”的特点
- 突出工具的易用性：无需编程，界面直观

### 3. 功能亮点展示
- 实时行情数据（强调数据准确性）
- 多维度智能筛选（价格、涨跌幅、成交量、基本面、行业）
- 一键生成专业报告（CSV/JSON格式）
- 响应式设计（电脑/手机都能用）

### 4. 使用场景案例
- 寻找低估值股票（PE<15, ROE>10%）
- 发现高成长性股票（当日涨跌幅>3%，成交量活跃）
- 避险筛选（稳定大盘股）
- 行业轮动筛选

### 5. 技术优势背书
- 全自动化运行（7×24小时不间断）
- 数据每10分钟刷新
- 开源项目，代码透明
- 未来规划：AI选股模型、量化回测

### 6. 社会证明与可信度
- 开发者背景：量化投资爱好者 + 开发者
- 工具开发动机：解决自身投资痛点
- 开源社区支持（GitHub stars、用户反馈）

### 7. 行动号召（转化关键）
- 提供立即体验链接（本地版 + 云端版）
- 引导访问GitHub仓库（增加项目可信度）
- 邀请反馈和功能建议
- 预告未来功能更新

## 文章结构要求
1. **标题**：吸引眼球，包含关键词“免费”、“智能选股”、“AI助手”，长度适中（15-25字）
2. **导语**：用问题或故事引出痛点，引发读者共鸣
3. **核心功能**：分点介绍，每个功能配使用场景
4. **使用教程**：步骤清晰，配截图或示例
5. **技术细节**：适度展示技术实力，增加可信度
6. **未来规划**：展现项目长期价值
7. **风险提示**：合规免责，体现专业性
8. **结语**：鼓励行动，留下深刻印象

## 写作风格指南
- **专业但亲切**：用投资者能懂的语言解释技术概念
- **数据驱动**：用具体数字证明工具价值（如“覆盖60+只股票”、“每10分钟刷新”）
- **故事性**：从个人经历出发，增加真实感
- **互动性**：使用“你”称呼读者，增加代入感
- **视觉化**：描述使用场景时让读者能“看到”效果

## 转化策略
1. **内链引导**：文中多次提到工具链接，方便点击
2. **价值前置**：先提供足够价值（免费工具），再引导行动
3. **社会认同**：展示用户好评或使用案例
4. **稀缺性暗示**：强调“完全免费”可能不会永远持续
5. **下一步明确**：告诉读者具体做什么（访问链接、下载报告、关注更新）

## 流量优化策略
1. **关键词布局**：自然融入“A股投资”、“股票筛选”、“量化分析”、“免费工具”等搜索词
2. **热门话题关联**：关联“人工智能投资”、“散户如何选股”、“价值投资”等知乎热门话题
3. **发布时间**：选择股市收盘后（下午3点后）或周末发布
4. **互动引导**：结尾提问引发评论，如“你最需要哪种筛选功能？”
5. **多平台分发**：文章可同步到雪球、集思录等投资社区

## 合规注意事项
1. 明确声明“不构成投资建议”
2. 强调“投资有风险，入市需谨慎”
3. 数据仅供参考，不保证准确性
4. 尊重版权，注明数据来源
